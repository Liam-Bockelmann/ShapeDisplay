SPECIAL CONTROLS
---------------------------------------------------
'c' = Clears the screen of any object

'z' = Toggles between wireframe mode and solid mode. Current shape and all future shapes will be displayed in
	that mode until it is changed with the z key. Starts on wireframe mode.


SHAPE CHOOSING CONTROLS
---------------------------------------------------
'1' = Display a cube

'2' = Display a tetrahedron

'3' = Display a low quality sphere approximation

'4' = Display a medium quality sphere approximation

'5' = Display a high quality sphere approximation

'6' = Display a quadric sphere

'7' = Display a quadric cone

'8' = Display a quadric cylinder

'9' = Display a low quality ellipsoid

'0' = Display a medium quality ellipsoid

'q' = Display a high quality ellipsoid

'w' = Display a low quality torus

'e' = Display a medium quality torus

'r' = Display a high quality torus


SHAPE TRANSFORMATION CONTROLS
--------------------------------------------------
'LEFT_ARROW_KEY' and 'RIGHT_ARROW_KEY' = Rotate the shape along the x axis

'UP_ARROW_KEY' and 'DOWN_ARROW_KEY' = Rotate shape along the y axis

'LEFT_MOUSE CLICK' = Scale the shape up and make it larger

'RIGHT_MOUSE_CLICK' = Scale the shape down and make it smaller
